# IR Kit — 日本語

**10人のIRチームがいるように投資家対応を — たとえあなたとAIエージェントだけでも。**

IR Kit はオープンソースでローカルファーストな投資家関係（IR）システムです：資本政策表とSAFE転換計算、財務ダッシュボード（バーンレート／ランウェイ）、投資家CRM、業界標準形式の月次アップデート、データルームのチェックリストと運用プレイブック — コーディングエージェント（Claude Code、Codex、Cursor…）と協働する前提で設計されています。

## クイックスタート

```bash
git clone https://github.com/howieyoung/ir-kit && cd ir-kit
node server.js        # Node 18+、依存ゼロ — インストールはこれだけ
# → http://127.0.0.1:2330
```

## 初日はふた言だけ：エージェントに「ir start」

1. アプリを起動（サンプル企業データですぐ動きます）。
2. リポジトリでエージェントを開く — AGENTS.md を自動で読み込みます。
3. **「ir start」**と伝える。エージェントがフォルダ一式（書類の受け箱 inbox を含む）を作成し、プライバシーのルールを説明し、以降を案内します。
4. 財務・資金調達・会社の書類をすべて inbox へ — SAFE、銀行明細、資本政策表、ピッチ資料、未整理のままで構いません。
5. `ir sort` が書類をデータルームの正しいカテゴリへ自動分類 — **いつでも再実行可能**な常設のファイリングサービスです。
6. あなたの同意のもと、エージェントが書類を読み取り、実際の資本政策表・SAFE台帳・月次財務を構築 — **すべての数字に出典（ファイル＋ページ）付き**。不明点は確認リストへ、推測は一切しません。
7. `ir check` が通り、あなたが数字を確認したら、ダッシュボードは実データに。書類を渡さなければサンプルデータのままです。

## プライバシー（ポリシーではなく設計）

- データはローカルのJSON（`data/`）と書類フォルダ（`ir-workspace/`）— **どちらも .gitignore 済み**。
- サーバーはデフォルトで localhost のみ。公開デモの入力は訪問者自身のブラウザにのみ保存。
- アカウント不要、クラウド不要 — 良いIRのために資本政策表をどこかにアップロードする必要はありません。
