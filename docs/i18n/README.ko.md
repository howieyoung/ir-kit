# IR Kit — 한국어

**10명의 IR 팀이 있는 것처럼 투자자 관계를 운영하세요 — 당신과 AI 에이전트뿐이라도.**

IR Kit은 오픈소스, 로컬 우선 투자자 관계(IR) 시스템입니다: 캡테이블과 SAFE 전환 계산, 재무 대시보드(번레이트/런웨이), 투자자 CRM, 업계 표준 형식의 월간 업데이트, 데이터룸 체크리스트와 운영 플레이북 — 코딩 에이전트(Claude Code, Codex, Cursor…)와 함께 쓰도록 설계되었습니다.

## 빠른 시작

```bash
git clone https://github.com/howieyoung/ir-kit && cd ir-kit
node server.js        # Node 18+, 의존성 제로 — 설치는 이게 전부
# → http://127.0.0.1:2330
```

## 첫날은 두 마디면 됩니다: 에이전트에게 "ir start"

1. 앱 실행(샘플 회사 데이터로 즉시 작동).
2. 저장소 폴더에서 에이전트 실행 — AGENTS.md를 자동으로 읽습니다.
3. **"ir start"**라고 말하세요. 에이전트가 모든 폴더(문서 수신함 inbox 포함)를 만들고, 프라이버시 규칙을 설명하고, 이후를 안내합니다.
4. 모든 재무/펀드레이징/회사 문서를 inbox에 넣으세요 — SAFE, 은행 명세서, 캡테이블, 피치덱, 정리 안 된 상태 그대로 괜찮습니다.
5. `ir sort`가 문서를 데이터룸의 올바른 카테고리로 자동 분류 — **언제든 재실행 가능**한 상시 파일링 서비스입니다.
6. 동의하에 에이전트가 문서를 읽고 실제 캡테이블, SAFE 장부, 월별 재무를 구축 — **모든 숫자에 출처(파일+페이지)**가 붙고, 불확실한 것은 확인 목록으로. 절대 추측하지 않습니다.
7. `ir check` 통과 후 숫자를 확인하면 대시보드가 실제 데이터로 바뀝니다. 문서를 제공하지 않으면 샘플 데이터가 유지됩니다.

## 프라이버시(정책이 아닌 설계)

- 데이터는 로컬 JSON(`data/`)과 문서 폴더(`ir-workspace/`) — **둘 다 .gitignore 처리**되어 공개 저장소로 새어 나갈 수 없습니다.
- 서버는 기본적으로 localhost에만 바인딩. 공개 데모의 입력은 방문자 브라우저에만 저장됩니다.
- 계정 없음, 클라우드 없음 — 좋은 IR을 위해 캡테이블을 어딘가에 업로드할 필요는 없습니다.
